Linux C/C++工程中可生成ELF、动/静态库文件的通用Makefile
======================================================


## 1 描述
一个*nix环境下的Makefile文件，支持编译C/C++，同时能够通过参数配置生成ELF目标文件、动态链接库(.so)和静态链接库(.a)文件。
